2. Dados tratados

Podemos tratar dados de contato enviados voluntariamente, dados de clientes contratantes e dados operacionais necessarios para configurar assistentes de IA, agendamentos e paineis.

3. Finalidade

Os dados sao usados para responder contatos, prestar servicos contratados, configurar solucoes digitais e cumprir obrigacoes legais.

4. Compartilhamento

Nao vendemos dados pessoais. Podemos compartilhar dados com provedores tecnicos necessarios para hospedagem, banco de dados, comunicacao e execucao do servico.

5. Seguranca

Adotamos medidas tecnicas e administrativas razoaveis para proteger os dados contra acesso indevido, perda, alteracao ou divulgacao nao autorizada.

6. Direitos do titular

Voce pode solicitar acesso, correcao, exclusao, portabilidade ou informacoes sobre o tratamento dos seus dados pelo email informado nesta pagina.

7. Atualizacoes

Esta politica pode ser atualizada periodicamente. A versao mais recente ficara disponivel nesta pagina.
